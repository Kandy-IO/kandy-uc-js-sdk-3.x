# @kandy-io/uc-sdk-3.x

## Install

### Using npm :

`$ npm install @kandy-io/uc-sdk-3.x`

#### Install a particular build :

To install a particular build branch or beta branch with the Github URL, run:

`$ npm install https://github.com/Kandy-IO/kandy-uc-js-sdk-3.x#<build number or "beta"> --save`

### In the browser :
```html
<script src="https://unpkg.com/@kandy-io/uc-sdk-3.x@3.13.0/dist/kandy.js"
  integrity="sha384-ya9FXY9OaST5cotwg3iwvSQaZz7rHmw6/xYMYRffWh9+cRay3RrT8ka1QYKCazcQ"
  crossorigin="anonymous">
</script>
```
