@kandy-io/uc-sdk

### Npm Install

#### Package name :

`$ npm install @kandy-io/uc-sdk`

#### Github url :

Install with the beta branch or a particular build with the Github URL

`$ npm install https://github.com/Kandy-IO/kandy-uc-js-sdk#<build number or "beta"> --save`
