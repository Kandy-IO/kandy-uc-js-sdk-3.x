# @kandy-io/uc-sdk

## Install

### Using npm :

`$ npm install @kandy-io/uc-sdk`

#### Install a particular build :

To install a particular build branch or beta branch with the Github URL, run:

`$ npm install https://github.com/Kandy-IO/kandy-uc-js-sdk#<build number or "beta"> --save`

### In the browser :
```html
<script src="https://unpkg.com/@kandy-io/uc-sdk@3.7.0/dist/kandy.js"
  integrity="sha384-z6+LUduNgl/TtfcHxNO17vNZ/1EDpQBVIB2gLENJ/MjGWQIMO7ziru542e2yB6KU"
  crossorigin="anonymous">
</script>
```
